# 廣岡商店 Webサイト作業メモ

このフォルダは、廣岡商店HPの静的HTML一式です。

## 主なページ

- `index.html`：酒小売向けトップページ
- `recipes.html`：家飲みレシピ一覧
- `recipe-submit.html`：家飲みレシピ投稿フォーム
- `business.html`：飲食店様・事業者様向けの裏導線ページ

## レシピ投稿フォーム

`recipe-submit.html` の投稿フォームは、既存GAS WebアプリURLへ送信する設定済みです。

GAS側には `gas-recipe-submit.gs` の内容を、既存のApps Scriptプロジェクトへ追記してください。

想定される保存先シート名は `レシピ投稿` です。存在しない場合はGAS側で自動作成します。

## GAS反映手順

1. 既存のApps Scriptを開く
2. `gas-recipe-submit.gs` の内容を追記する
3. Webアプリのデプロイを更新する
4. `recipe-submit.html` からテスト投稿する
5. スプレッドシートに `レシピ投稿` シートが作成され、投稿内容が入ることを確認する

## 注意

投稿内容は確認後に掲載する前提です。20歳未満の方の投稿、飲酒を過度に促す内容、運転前後の飲酒につながる内容は掲載対象外です。
